Cara menjalankan "JUMP HELI JUMP"

1. Run program
2. Anda akan diminta memilih warna untuk helikopter
   Pilih warna yang diinginkan dengan memasukkan angka dari 0-15
   Urutan angka pertama dimulai dari warna kuning, bertambah ke kanan 
   lalu warna pink ke kanan.
3. Setelah memilih warna yang diinginkan, mulai bermain dengan memasukkan -1
4. Permainan dimulai, tugas anda adalah menggerakkan helikopter agar 
   terhindar dari awan dan bangunan dengan tombol 'i' untuk keatas dan 'k'
   untuk kebawah
5. Permainan akan berakhir jika anda mengenai awan atau bangunan 
   (atau jika programnya berhenti dengan sendirinya ._.)
   
Selamat Bermain
